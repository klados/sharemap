# sharemap
share a map with your friends. Cooperate in real time

## Insert databse
start your mysql service. Then load the schema from the mysql prompt
```
source database.sql
```

## Start the app
```
python app.py
```

## Page url
```
http://localhost:5000/sharemap/
```
